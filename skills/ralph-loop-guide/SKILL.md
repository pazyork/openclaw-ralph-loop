# RALPH-LOOP Guide

## Welcome to RALPH-LOOP

RALPH-LOOP helps you execute long-running, continuous, single-focus tasks. Before we begin, I would like to understand your task.

## Socratic Questioning

Please describe the task you want to accomplish. You can start like this:

"I want to use RALPH-LOOP to..."

## Question Checklist

After you describe it, I will guide you through clarifying:

1. **Task Goal** - What do you want to deliver?
2. **Why Multiple Rounds** - Why can't this be completed in a single round?
3. **Round Range** - Minimum rounds? Maximum rounds?
4. **Stage Requirements** - Are there different stages requiring different context isolation?
5. **Constitutional Constraints** - What is absolutely forbidden for the Agent?
6. **Progress Updates** - How often should I report progress (every N rounds)?
